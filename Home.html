My Journey


Curiosity sparked my journey into bespoke web development, evolving into a passion for building intuitive, visually striking websites. Over time, I refined my skills in coding, design, and UX to create digital experiences that truly connect with users.

Now, as a freelance web developer and designer based in Peebles, I thrive on turning ideas into reality. Every project is a chance to blend creativity with functionality, crafting sites that not only look great but also serve a purpose.


  • Highlighting a local gift shop featuring handcrafted Scottish products, showcasing artisans, traditions, and a café that complements the experience.
    A dynamic, data driven e-commerce site. Technologies: HTML, CSS, JS, PHP, SQL, JSON, AJAX, MYSQL

  • Showcasing the best of cycling—explore local trails, discover essential ride tips, and browse top-quality bikes and gear, all available in-store to fuel your next adventure.
    Technologies: HTML, CSS, JS

  • A cozy Edinburgh café serving artisan coffee, freshly baked cupcakes, muffins, and a variety of delicious vegan treats in a warm, inviting setting.
    The main focus for this application was to demonstrate foundational React JS knowledge.
    Technologies: HTML, CSS, React JS, MUI

  • A local cinema with five venues, featuring film screenings, special events, and a welcoming atmosphere for movie lovers in the community.
    Technologies: HTML, CSS, JS, PHP, SQL, MYSQL
    The main focus for this project was to create a dynamic, data driven application.

  • An organic farm offering locally produced honey and a farm shop stocked with fresh, organic products, connecting visitors to sustainable farming and natural goodness.
    Technologies: HTML, CSS, Bootstrap
