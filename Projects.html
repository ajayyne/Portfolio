PROJECTS


I've developed a number of web appplications through my HND at Edinburgh College. Below is a collection of my work.

ORGANIC ME



Organic Me was my first web project ever. It is a simple static website to advertise a local Borders farm and their organic produce

The project utilized the Bootstrap framework to ensure a responsive design across many devices.

Users can browse products, find the farm shop information, send queries via a contact form, view the farm on the google maps API, and view courses on offer.


Bike King



Bike King, located in the Scottish Borders, sells and rents road, mountain, and hybrid bikes. They offer a number of repair services and advertise local biking trails.

The project was my first website built with JavaScript to store bike images and details.

Users can search for bikes using the search filters provided, browse apparel and gear, find informaiton on services and repairs, pricing for rentals and local trail locations. They can contact the business, find social media links and find opening hours of the store.


LIMELIGHT CINEMA



Limelight cinema has various locations around the UK, however - this site was built with a focus on the midlothian area.

Built using PHP and SQL to generate dynamic movie content - the website was my first project using backend languages.

Users can create an account, browse currently showing films, find showtimes for specific venues and dates, and book tickets.
